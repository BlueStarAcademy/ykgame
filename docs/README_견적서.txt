YK건기 시뮬레이터 개발 견적서 (재생성본)
=====================================

권장 파일 (한글/워드에서 열기)
------------------------------
1) docs/YKGame_Dev_Estimate_2026.docx
   - Microsoft Word / 한글(한컴오피스) 모두에서 안정적으로 열립니다.
   - 한글에서 .hwp 가 필요하면: 파일 > 다른 이름으로 저장 > HWP 문서(*.hwp)

2) docs/YKGame_Dev_Estimate_2026.hwpx
   - 한글 개방형 형식(HWPX). 한컴오피스에서 엽니다.
   - 이전 파일은 생성기/파일명 인코딩 문제로 손상될 수 있어 재작성했습니다.
   - ASCII 파일명을 사용해 Windows에서도 경로 깨짐이 없습니다.

이전 손상 파일
--------------
- 한글 파일명 HWPX는 생성·저장 과정에서 깨질 수 있어 삭제되었습니다.
- 위 두 파일만 사용하세요.

권유안 요약
----------
- 개발 3개월 / 6,000만원 (분할)
- 유지보수 12개월 / 월 200만원
- 이후 6개월 단위 연장
- 추가 개발은 별도 비용

재생성
------
cd docs/estimate-build
npm install
node generate-estimate-v2.mjs
